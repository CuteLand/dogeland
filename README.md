9# DogeLand-App
Run Linux on android.
开箱即用的Android运行Linux解决方案
## 快速上手
本应用程序安装选定的 GNU/Linux 发行版，并在 chroot / proot 容器中运行。
 使用步骤：  
  1. 给本应用 超级用户（Root） 权限。
  2. 正常连接至互联网。
  3. 导入Linux系统包
  4. 设置相关参数
  5. 点击"启动"选项按钮来启动容器。
  6. 通过 CLI、SSH、VNC 等方式连接容器
           
本应用同时兼容LinuxDeploy安装的系统，只设置安装路径与命令行即可使用。
